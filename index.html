Fr

Fr -Stand for fraction - and the fr unit represents a fraction of the available space in the grid. set the value to 1fr all around mean all available space will share the same fraction.This method has responsiveness buildin , if the container getting smaller and smaller the column evebtually get stretch out. You can applied fr twice in the same column . Ex: 2fr

/* Challenge: 1. Set up the "main" element as a grid container. 2. Specify a template of one column with 1fr width. 3. Place 1em of space between each grid item. */ display: grid; grid-template-columns: 1fr; gap: 1em;
