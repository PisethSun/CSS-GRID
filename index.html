CSS Grid – Study Notes

1. The fr Unit

fr stands for fraction. It represents a fraction of the available space in a grid container.

When you set 1fr on all columns, it means all columns share the space equally. This makes your layout naturally responsive: as the container gets smaller, the columns shrink with it.

/* Challenge: 1. Set up the "main" element as a grid container. 2. Specify a template of one column with 1fr width. 3. Place 1em of space between each grid item. */ main { display: grid; grid-template-columns: 1fr; gap: 1em; }
You can also use values like 2fr, 3fr, etc. For example, 2fr 1fr means the first column gets twice as much space as the second.

3. Example Layout with grid-row and grid-column

Example using header, nav, main and footer:

/* Layout challenge example */ header { background-color: palegoldenrod; grid-column: span 3; } nav { background-color: lightcoral; grid-column: span 1; grid-row: span 3; } main { background-color: lightgreen; grid-row: span 3; grid-column: span 2; } footer { background-color: gold; grid-column: span 3; }

Another challenge example:

/* Make the layout match the design */ main { display: grid; grid-template-columns: 1fr 1fr; } .technology { grid-column: span 2; } .nasa { grid-row: span 2; } .health { grid-column: span 2; }
grid-template-columns: 1fr 1fr; tells the <main> element (with display: grid) to create two equal-width columns.

4. The repeat() Function

Basic usage:

.grid-container { display: grid; gap: 0.5em; grid-template-columns: repeat(8, 1fr); /* repeat(how many times, what to repeat) */ border: 2px solid black; }

You can also mix repeat() with other values:

grid-template-columns: repeat(7, 1fr) auto;

Defining both rows and columns with grid-template shorthand:

.grid-container { display: grid; /* 5 equal rows, 12 equal columns */ grid-template: repeat(5, 1fr) / repeat(12, 1fr); }

5. Explanation: grid-template-columns: repeat(12, 1fr);

This line creates a 12-column grid with equal-width columns.

  • repeat(12, ...) → make 12 columns
  • 1fr → each column takes 1 fraction of the available space
  • Because all columns use 1fr, each column has equal width.

.twelve-col-grid { display: grid; grid-template-columns: repeat(12, 1fr); gap: 0.5rem; }
This kind of 12-column grid is very common in modern layouts. You can then place items like: grid-column: span 3; (3 columns wide), grid-column: span 6; (half the grid), etc.

7. Using grid-template-areas with Cards

1. Assign each card a grid-area name:

.technology { grid-area: tech; } .environment { grid-area: envi; } .nasa { grid-area: nasa; } .physics { grid-area: phys; } .health { grid-area: heal; }

2. Use grid-template-areas to define the layout (mobile example):

main { display: grid; gap: 1em; grid-template-columns: 1fr; grid-template-areas: "tech" "envi" "nasa" "phys" "heal"; }
